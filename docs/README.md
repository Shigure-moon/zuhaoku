## 🚀 快速开始

### 新成员入门
1. 阅读 [DEVELOPMENT_GUIDE.md](./DEVELOPMENT_GUIDE.md) 了解项目概况
2. 根据角色选择：
   - 前端开发 → [FRONTEND_DEVELOPMENT.md](./FRONTEND_DEVELOPMENT.md)
   - 后端开发 → [BACKEND_DEVELOPMENT.md](./BACKEND_DEVELOPMENT.md)
3. 查看 [ARCHITECTURE.md](./ARCHITECTURE.md) 了解系统架构
4. 按照文档完成环境搭建
